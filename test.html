建立博客

建立github相应的仓库

建立这个博客的时候，关于git的坑踩了特别多。这个放到接下来一篇来写。
首先我们在git上建立一个仓库，名字为yourname.github.io

填好名字确认之后，我们的博客空间就已经建立了。

hexo初步搭建

安装了node.js之后，在你相应的根目录下执行下面代码来安装hexo

npm install -g hexo

git init啦之类的。hexo安装好之后建立hexo博客框架

hexo init

生成静态页面

hexo g

启动本地服务器，在浏览器中输入localhost:4000可进入生成的静态界面

hexo s


接下来打开博客目录下的_config.yml文件，建立与github库的关联

配置完成之后记得hexo d来部署hexo,若报错执行npm install hexo-deployer-git --save
再正常的提交代码，然后你就可以去你的博客yourname.github.io观摩了。可能会有个十分钟的延迟。
